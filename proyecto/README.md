# Mini Blog en Django

Proyecto realizado en Django utilizando el patrón MVT.

## Modelos
- Autor
- Categoria
- Post

## Funcionalidades
- Crear autores
- Crear categorías
- Crear posts
- Buscar posts por título

## Orden recomendado para probar
1. Crear un autor
2. Crear una categoría
3. Crear un post
4. Buscar el post
